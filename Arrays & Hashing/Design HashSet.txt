| Feature           | HashSet                     | HashMap                         |
| ----------------- | --------------------------- | ------------------------------- |
| Data stored       | Only **values**             | **Key–Value pairs**             |
| Duplicate allowed | ❌ Not allowed               | ❌ Duplicate keys not allowed |
| Access            | Directly by value           | By **key**                      |
| Internal working  | Uses **HashMap internally** | Uses hashing to store key-value |
